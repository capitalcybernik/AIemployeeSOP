# Content Library - Proof Points Collection

**Name:** Content Library  
**Description:** Builds and maintains a comprehensive library of proof points including case studies, testimonials, statistics, awards, certifications, media mentions, and before/after results for use in all marketing content.

## What This Skill Does

The Content Library skill creates a structured repository of credibility-building content that the AI Employee can reference when creating any marketing materials. This includes case studies, client testimonials, industry statistics, company awards, certifications, media mentions, and measurable results. Each entry is tagged with source, date, context, and which services it supports, making it easy to find the right proof point for any content creation task.

This library serves as the ammunition for all persuasive content, ensuring the AI Employee always has compelling evidence to support claims and build trust with prospects.

## How to Collect Information

### Discovery Questions for Client

#### Existing Proof Points
1. What are your biggest client success stories?
2. Do you have written testimonials or reviews from customers?
3. What specific results have you achieved for clients (numbers, percentages, dollar amounts)?
4. What industry awards or recognition have you received?
5. What certifications, credentials, or qualifications do you hold?
6. Have you been featured in any media (podcasts, interviews, articles)?
7. What statistics about your industry or approach do you frequently cite?

#### Context and Usage
1. Which services or products does each proof point support?
2. What stage of the customer journey is each most effective for (awareness, consideration, decision)?
3. Are there any restrictions on how certain testimonials or case studies can be used?
4. Which proof points are your strongest for overcoming common objections?

#### Source Documentation
1. Can you provide original sources for all statistics you use?
2. Do you have written permission to use client names in case studies?
3. What format are your testimonials in (video, written, recorded audio)?
4. Do you have high-resolution images of awards or certificates?

## How to Use in Daily Operations

### Content Creation Workflow
1. Before writing any marketing content, scan the Content Library for relevant proof points
2. Match proof points to the content type (blog post, sales page, social media)
3. Use the tagging system to find proof points specific to the service being promoted
4. Include 2-3 proof points per piece of content when possible

### Objection Handling
1. Identify common objections for each service
2. Map specific proof points to counter each objection
3. Reference the library when creating FAQ content or sales scripts
4. Update responses as new proof points are added

### Social Proof Strategy
1. Rotate testimonials across different content pieces
2. Use fresher proof points more frequently
3. Match proof points to audience segments (industry, company size, etc.)
4. Create proof point calendars for consistent usage

### Maintenance Tasks
1. **Monthly:** Add new testimonials and results as they come in
2. **Quarterly:** Verify all statistics are still current and update sources
3. **Annually:** Audit usage rights and permissions for all content
4. **Ongoing:** Tag new entries consistently for easy retrieval
